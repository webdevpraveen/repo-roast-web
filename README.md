# 🔥 Repo Roast Frontend

The visual interface for Repo Roast. A sleek, terminal-inspired landing page built with Next.js 15, styled with a "Hellfire" aesthetic.

## ✨ Features

- **Hellfire UI**: Dynamic fire particles, CRT noise effects, and a brutalist design.
- **Real-time Roasting**: Interactive terminal-style loading states and instant feedback.
- **Stat Dashboards**: Visualizes repo metrics like stars, bad commits, and open issues.
- **Social Sharing**: One-click sharing to Twitter/X.
- **Extreme SEO**: Optimized for web crawlers with full OpenGraph support and JSON-LD.

## 🛠 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Vanilla CSS (Custom Keyframes & Animations)
- **Typography**: Bebas Neue & Space Mono
- **Deployment**: Vercel
